// HOW IT WORKS

Three quiet steps.

01

Write a template

One prompt with a {} where the name goes. Paste your list of subjects — poses, products, names — one per line or from a PDF.

02

Pick your models

Send the batch to ChatGPT, Gemini, or both at once. Set how many parallel workers each provider runs, then start the queue.

03

Watch & collect

The monitor fills in real time. Completed images auto-download, failures retry, and rate limits never derail the run.
